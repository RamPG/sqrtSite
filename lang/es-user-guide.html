1 INTRODUCCIÓN

  1. Ámbito de aplicación

Ámbito de aplicación del producto de software: cálculo de la raíz cuadrada de expresiones matemáticas con números y números.

  1. Breve descripción de las características 

El usuario tiene la capacidad de trabajar con operaciones aritméticas básicas, funciones trigonométricas, logaritmos, números complejos, módulo numérico, constantes matemáticas, raíces numéricas, exponenciación. 

  1. Nivel de formación del usuario

Para tener éxito con el producto, el usuario debe tener experiencia en el funcionamiento de una computadora personal/Teléfono inteligente y uno de los sistemas operativos Windows 7+, Windows 7 y superior, distribuciones de Linux con actualizaciones posteriores a 2009, masOS 10.10 y superior, Android 5 y superior, iOS 9 y superior. Ser capaz de manejar los navegadores web. Es necesario saber qué es una expresión matemática con números. Comprender qué son las funciones trigonométricas, los logaritmos, los números complejos, el módulo de números, las constantes matemáticas, la raíz del número, la exponenciación. También conocer las prioridades de las operaciones en matemáticas.

  1. Lista de documentación de funcionamiento que debe familiarizarse con el usuario

Para una mejor comprensión de cómo se realizan los cálculos, debe Leer la documentación de la biblioteca mathjs con la que se ejecuta el programa.

  1. PROPÓSITO Y CONDICIONES DE APLICACIÓN
    1. Actividades, funciones para las que está diseñada esta herramienta de automatización

El producto de software automatiza el cálculo de la raíz cuadrada de expresiones matemáticas con números y números.

  1. Condiciones en las que se cumple (ejecución, ocurrencia) la aplicación de la herramienta de automatización de acuerdo con el propósito previsto (por ejemplo, tipo de computadora y configuración de equipo, entorno operativo y software de todo el sistema, información de entrada, soportes de datos, base de datos, requisitos de capacitación, etc.) 

Para que el programa funcione correctamente, es necesario cualquiera de los navegadores (Firefox, Google Chrome, Opera, Safari, Internet Explorer, etc.) que sea compatible con los sistemas operativos móviles(Android 5 y superior, iOS 9 y superior) o de Escritorio (Windows 7 y superior, distribuciones de Linux con actualizaciones posteriores a 2009). Para trabajar en la interfaz gráfica de usuario, se utiliza un manipulador de tipo "ratón" (PC), un Sensor (Teléfono inteligente). También es necesario que el navegador permita el uso de JavaScript.

  1. PREPARACIÓN PARA EL TRABAJO
    1. Composición y contenido del medio de distribución

En la carpeta principal del programa se encuentran los archivos index.html, build.js, style.css, así como la subcarpeta lang, donde se presentan los archivos html para apoyar el multilenguaje del producto.

Abrir el archivo index.html usando uno de los navegadores web que está instalado en su sistema operativo.

El estado del sistema no requiere que el usuario realice ninguna verificación adicional, si el navegador web ha abierto el archivo index.html y muestra la interfaz gráfica del programa, el sistema está en modo normal.

4. DESCRIPCIÓN DE LAS OPERACIONES

 

  1. Campo de entrada

Es una representación gráfica de las teclas pulsadas por el usuario.

  1. Campo de salida

Representa una representación gráfica del resultado de una expresión que el usuario ha introducido.

  1. Introducción del número de decimales 

Representa un campo de entrada en el que el usuario puede ingresar un número entre 1 y 100 que captura la visualización del número de decimales. ATENCIÓN: cuando se trabaja con números complejos, la precisión del número y el número de caracteres pueden ser mucho menores que el número introducido en el campo de entrada de decimales.

4.4 Botones numéricos (0-9), así como constantes matemáticas (exponente (e), número PI ()), botón para introducir una medida de grado (•), paréntesis para la prioridad de operaciones ' ( ' , ')', unidad imaginaria ('i'), coma','

Al hacer clic en uno de los botones de la interfaz gráfica de usuario, se muestra y se agrega a la línea de entrada. ADVERTENCIA: los ceros principales antes del número se omiten.

4.5 Funciones (log, sin, cos, abs, tg, ctg,  √, ^)

) log. Al hacer clic, la función se muestra en la interfaz gráfica ('log (' ), debe introducir el número y la base del logaritmo separados por comas y cerrar el paréntesis (log (32, 2)), si no se ingresa el segundo argumento, la base del logaritmo será exponente.

) sin/cos/tg/ctg. Al hacer clic, se muestra una función en la interfaz gráfica de usuario (por ejemplo, 'sin (')), debe introducir un número o agregar una medida de grado con el botón de entrada de medida de grado y cerrar el paréntesis. Ejemplo: sin (3) o sin(30 deg).

) abs, √, ^. Al hacer clic, se muestra la función en la interfaz gráfica de usuario. Ejemplo: sqrt (. Debe introducir un número y cerrar el paréntesis). Ejemplo: abs (-3).

4.6 Operaciones matemáticas (+, -, /, *)

Al hacer clic en uno de los botones de la interfaz gráfica de usuario, se muestra y se agrega a la línea de entrada. ATENCIÓN: para que el programa funcione correctamente, es recomendable no omitir el signo de multiplicación (*).

4.7 Botones para trabajar con el campo de entrada (AC, C).

) C. Al hacer clic en el botón se elimina el último carácter del campo de entrada.

) AC. Al hacer clic en el botón, el campo de entrada se borra.

4.8 Botón de salida de resultado (=)

Muestra el resultado ingresado en el campo de entrada en el campo de salida.

4.9 Botones para cambiar el idioma del producto

Cambiar el idioma del producto al hacer clic en función del botón seleccionado.

4.10 Botón de Servicio al cliente

Redirige al usuario al canal de soporte.

5. EMERGENCIAS

En caso de emergencia, póngase en contacto con nuestro Servicio de atención al cliente. Para hacer esto, debe tener una cuenta o registrarse en la aplicación Telegram, instalar la aplicación o usar su versión web. A continuación, haga clic en el botón "soporte Técnico", que se encuentra en la parte inferior de la página del programa, se le redirige a nuestro bot de soporte multilingüe. Allí puede obtener respuestas a las preguntas más frecuentes o llamar a un operador para que le responda en línea.
