1 INTRODUCTION

  1. Scope of application

Scope of application of the software product: calculation of the square root of mathematical expressions with numbers and numbers.

1.2 Brief description of the features

The user has the opportunity to work with basic arithmetic operations, trigonometric functions, logarithms, complex numbers, the modulus of a number, mathematical constants, roots of a number, exponentiation. 

1.3 User training level

To successfully work with the product, the user must have experience operating a personal computer/smartphone and one of the operating systems Windows 7+, Windows 7 and higher, Linux distributions with updates after 2009, masOS 10.10 and higher, Android 5 and higher, iOS 9 and higher. Be able to handle web browsers. It is necessary to know what a mathematical expression with numbers is. Understand what trigonometric functions, logarithms, complex numbers, the modulus of a number, mathematical constants, the root of a number, exponentiation are. Also know about the priorities of operations in mathematics.

1.4 The list of operational documentation that the user needs to read

For a better understanding of how calculations occur, it is necessary to familiarize yourself with the documentation of the mathjs library, with which the program works.

4. DESCRIPTION OF OPERATIONS

 

  1. Input field

It is a graphical representation of the keys pressed by the user.

  1. Output field

It is a graphical representation of the result of the expression that the user entered.

  1. Entering the number of decimal places

It is an input field in which the user can enter a number in the range from 1 to 100, which fixes the display of the number of decimal places. ATTENTION: when working with complex numbers, the accuracy of the number and the number of characters may be much less than the number entered in the decimal place input field.

4.4 Number buttons(0-9), as well as mathematical constants(exponent (e), Pi (𝜋)), a button for entering a degree measure (•), brackets for the priority of operations ‘ ( ‘ ,‘)’, imaginary unit ('i’), comma‘, ’

When you click on one of the buttons on the graphical interface, they are displayed and added to the input line. ATTENTION: leading zeros before the number are omitted.

4.5 Functions (log, sin, cos, abs, tg, ctg,  √, ^)

А) log. When you click, the function is displayed on the graphical interface ('log (‘), you must enter a number and the base of the logarithm separated by a comma and close the bracket (log(32, 2)), if the second argument is not entered, then the base of the logarithm will be equal to the exponent.

Б) sin/cos/tg/ctg. When pressed, the function is displayed on the graphical interface (for example ‘ ' sin (‘), you need to enter a number or add a degree measure using the enter degree measure button, and also close the bracket. Example: sin(3) or sin(30 deg).

В) abs, √, ^. When clicked, the function is displayed on the graphical interface. Example: sqrt(. You must enter a number, and also close the bracket). Example: abs (-3).

4.6 Mathematical operations (+, -, /, *)

When you click on one of the buttons on the graphical interface, they are displayed and added to the input line. ATTENTION: for the correct operation of the program, it is advisable not to omit the multiplication sign(*).

4.7 Buttons for working with the input field (AC, C).

А) C. When you click on the button, the last character is removed from the input field.

Б) AC. When you click on the button, the input field is cleared.

4.8 The result output button (=)

Outputs the result entered in the input field to the output field.

4.9 Buttons for changing the product language

Change the language of the product when pressed, depending on the selected button.

4.10 Support Service button

Redirects the user to the support channel.
